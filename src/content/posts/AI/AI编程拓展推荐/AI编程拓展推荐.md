# 优质AI编程扩展&工具 全文总结
本文按**MCP服务器、IDE扩展插件、Agent Skills、跨编辑器通用工具、AI效率工具箱**五大类，系统化分享全套AI编程提效工具，打通AI联网、编码、自动化、文档处理全流程，大幅提升代码质量与开发效率。

## 一、MCP 服务器（Model上下文协议，AI万能外接接口）
核心作用：给本地AI编程工具（Cursor/Claude Code）解锁联网、浏览器、云服务、仓库权限，打破模型训练数据限制。
1. **Firecrawl MCP**
   能力：抓取单页/整站/多层递归网页，解析动态渲染内容、官方文档、技术博客。
   用途：让AI直接读文档、查参考资料、分析竞品页面，替代复制粘贴/手写爬虫。
2. **Crawl4AI（平替）**
   大模型友好爬虫，自带MCP能力，轻量化网页解析备选。
3. **Brave Search MCP**
   能力：隐私化全网搜索，支持网页/图片/新闻检索+AI结果摘要。
   用途：查最新技术报错、框架新特性、解决方案，无搜索记录追踪。
4. **Context7**
   能力：精准拉取各框架**指定版本最新官方文档**。
   用途：解决AI知识过时问题，避免废弃API、老旧语法，保证代码可运行。
5. **Web to MCP（Chrome扩展）**
   能力：捕获网页DOM、CSS样式、交互逻辑，自动生成复刻提示词。
   用途：看到优质UI组件，一键让AI复刻同款前端代码，比截图精准百倍。
6. **Chrome DevTools MCP**
   能力：AI直连Chrome，操控开发者工具、监控网络、元素定位、性能排查、自动填表测试。
   用途：前端页面调试、性能优化、自动化功能测试、线上问题排查。
7. **EdgeOne Pages MCP（腾讯云）**
   能力：项目一键打包、上传、全球CDN部署，自动HTTPS。
   用途：个人前端/静态项目快速上线，一键生成可访问链接。
8. **COS MCP（腾讯云对象存储）**
   能力：AI读写云端存储桶文件，管理图片、规范文档、静态资源。
   用途：团队共享项目规范、公共资源，云端文件协同开发。
9. **GitHub MCP**
   能力：操控代码仓库，查Star、提交记录、代码变更、创建Issue/PR、安全扫描。
   用途：AI辅助代码管理、开源项目维护、提交记录分析。

## 二、IDE 必备扩展（VS Code / Cursor 通用）
主打代码规范、调试提、Git协作、AI编码体验优化。
1. **Claude Code 插件**
   侧边栏可视化对话，代码实时Diff对比，多会话并行，擅长重构、修Bug、全栈功能开发。
2. **GitLens**
   行级代码溯源，查看作者、提交记录、PR关联，自带AI生成提交信息、变更解读。
3. **Office Viewer**
   编辑器内直读MD/Word/Excel/PDF/SVG，无需切换第三方阅读器。
4. **ESLint**
   实时检测语法错误、代码不规范写法、潜在逻辑漏洞，AI代码一键合规修复。
5. **Prettier**
   全局代码格式化，保存自动排版，统一引号、缩进、换行，解决团队代码风格冲突。
6. **Error Lens**
   错误/警告行内直接展示，不用悬浮查看，快速定位代码问题。
7. **Console Ninja**
   编辑器内展示前端`console`日志、接口请求、错误堆栈，脱离浏览器F12调试。

## 三、Agent Skills 技能库（AI专属预制工作流，跨工具通用）
核心：不用写长提示词，安装即解锁专业能力，Claude/Cursor全适配，分为**工具、平台、场景化技能**。
### 1. 技能管理工具
- skills CLI / find-skills：命令行一键搜索、安装、管理技能包
- Skill Creator：自定义制作专属AI技能
- Skill Seeker：网页/文档/Git仓库自动转为AI技能
- everything-claude-code：一站式配齐MCP/技能/配置，开箱即用

### 2. 官方基础技能
- document-skills：AI处理PDF/Excel/Word/PPT全办公文档
- example-skills：测试、动图、主题定制、网页测试通用能力

### 3. 前端开发核心技能
- frontend-design（Anthropic官方）：告别流水线UI，生成高辨识度生产级界面
- vercel-react-best-practices：约束React/Nest代码规范、性能优化、反模式规避
- vue-skills（尤雨溪团队维护）：Vue3+Vite+Pinia+Vitest全套最佳实践
- ui-ux-pro-max：专业UI/UX设计能力，自动优化配色、布局、响应式
- web-design-guidelines：统一排版、间距、色彩规范

### 4. 后端&数据库技能
- supabase-postgres-best-practices：SQL优化、索引设计、数据库规范
- stripe/ai：支付业务代码规范、接口最佳实践

### 5. 测试&安全技能
- trailofbits/skills：合约审计、漏洞检测、代码安全审查
- audit-website：网站安全、隐私泄露、性能多维度审计

### 6. 自动化&浏览器技能
- browser-use / agent-browser：AI模拟真人点击、填表、截图、批量爬取、E2E测试

### 7. 内容创作技能
- remotion-dev：AI生成编程演示动画、短视频
- humanizer：弱化AI文风，生成真人化文案
- baoyu-skills：公众号、小红书、PPT一键创作

### 8. 项目架构技能
- planning-with-files：文件化长期记忆，解决AI上下文丢失，适合长线复杂项目
- superpowers：TDD测试驱动、需求拆解、代码审查，适合大型项目开发

### 9. 第三方平台技能
- Notion Skills：AI管理笔记、会议纪要、知识库
- seo-audit：网站SEO优化、流量提升分析

### 10. 优质技能平台
鱼皮AI导航、skills.sh、skillsmp：一键查热门技能，免费安装复用

## 四、跨编辑器通用开发框架（规范驱动开发）
适合团队/中大型项目，约束AI开发流程，减少返工。
1. **Spec-kit（重流程）**
   七步标准化开发：定项目准则→写需求规范→澄清边界→技术方案→任务拆解→代码质检→编码落地。
   作用：强约束AI行为，需求、设计、代码完全对齐，适合从0新建项目。
2. **OpenSpec（轻量）**
   简化版规范开发：变更提案→审核→代码实现→归档同步。
   作用：适合已有项目迭代，轻量化维护文档与代码同步。
3. **AIChat 命令行工具**
   自然语言转Shell命令，记住打包、部署、文件操作等复杂指令，提升终端操作效率。

## 五、AI 全能效率工具箱（独立开源工具，补齐AI短板）
### 1. 文档解析工具（AI只读文本，这类工具实现多格式解析）
- MarkItDown（微软）：PDF/图片/视频/网页全能转Markdown，日常通用
- MinerU：论文专用，解析复杂排版、数学公式、表格
- Docling：复杂Word/PPT/图文混排文档精准还原结构

### 2. 音视频处理
- whisper.cpp：本地离线语音转文字，会议录音、视频字幕生成
- yt-dlp：全平台视频/音频下载，抓取素材
- FFmpeg：AI自动生成剪辑、转码、压缩、格式转换命令

### 3. 自动化与集成
- BrowserUse：AI浏览器自动化，批量操作、数据采集
- Composio：一键对接1000+平台（GitHub/邮箱/Notion），免重复写对接代码

### 4. 记忆增强
- Mem0：AI持久化记忆，记住你的技术栈、项目习惯、历史需求，新开对话不遗忘

## 整体总结
整套方案形成闭环：
1. **MCP** 负责让AI连外网、浏览器、云服务；
2. **IDE插件** 负责本地代码规范、调试、协作；
3. **Agent Skills** 给AI预装各行业专业能力，不用重复提示；
4. **规范框架** 管控项目开发流程，适合团队协作；
5. **效率工具箱** 解决文档、音视频、自动化、记忆等刚需短板。
   全栈开发者、前端、后端、测试、内容创作者都可按需搭配，直接落地提升AI编程生产力。
